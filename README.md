# 주요기능
+ 컴퓨터 모니터를 제어하는 3가지 모드가 있고 버튼을 눌러서 모드를 변경할 수 있다.
  + 초기상태 : 모니터 화면이 바탕화면을 보이게 하는 기능을 한다.
  + 버튼을 한 번 눌렀을 시 : ALT+TAP 기능을 한다.
  + 버튼을 두 번 눌렀을 시 : 컴퓨터를 종료 시키는 기능을 한다.
